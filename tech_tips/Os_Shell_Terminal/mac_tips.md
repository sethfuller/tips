### Shortcuts

| Keys                         | Description                   |
|------------------------------|-------------------------------|
| **\<Control>-\<Command>-F**  | Toggle Full Screen            |
| **\<Control>-\<Down Arrow>** | Show only current application |
| **\<Control>-\<Up Arrow>**   | Show all windows              |
