| Keys                              | Description                           |
|-----------------------------------+---------------------------------------|
| *<Command>-<left_arrow>*          | Next Tab                              |
| *<Command>-<right_arrow>*         | Previous Tab                          |
| *<Command>-<number>*              | Go to Tab Number                      |
| *<Command>-<Option/Alt>-<number>* | Go to Window Number (by opened order) |
| *<Command>-T*                     | New Tab                               |
| *<Command>-N*                     | New Window                            |
